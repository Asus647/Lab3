# Приложение для изучения языков

Приложение для изучения иностранных языков с графическим интерфейсом на Python.

## Функции

- Добавление и удаление слов с переводом
- Указание языка и сложности слова
- Отслеживание прогресса изучения
- Визуализация статистики с помощью графиков
- Логирование всех действий
- Управление через удобный графический интерфейс

## Установка и запуск

### Требования
- Python 3.8+
- Установленные зависимости
